# Walrus Simulator

Open-source simulation framework to test explicit assumptions about energy, materials, institutions, and emergent superorganism dynamics.

## Quality Gates

- Lint: `ruff`
- Types: `mypy --strict`
- Tests: `pytest`
- Coverage threshold: 90%

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e '.[dev]'
make check
```
