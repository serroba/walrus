# Plan 05: Open Source and Governance

## Goal

Make contribution, review, and model debate first-class features.

## Deliverables

1. Contribution guide with modeling RFC process.
2. Assumption change template requiring expected directional effects.
3. Benchmark suite for performance + scientific regressions.
4. Versioned scenario catalog with changelogs.
5. Governance model (maintainers, reviewers, release cadence).

## Community Design Rules

1. Separate normative claims from positive/mechanistic claims.
2. Require uncertainty annotation for all public charts.
3. Prefer small composable model modules over monoliths.
