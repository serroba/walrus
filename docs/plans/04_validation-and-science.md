# Plan 04: Validation and Scientific Method

## Goal

Define a transparent protocol for validating assumptions and outcomes.

## Validation Layers

1. Unit validation: accounting identities and invariants.
2. Mechanism validation: local behavior against known empirical ranges.
3. System validation: stylized macro facts reproduced.
4. Forecast stress tests: sensitivity and uncertainty envelopes.

## Deliverables

1. Hypothesis registry (`docs/hypotheses/*.md`).
2. Experiment registry with preregistered metrics.
3. Calibration and sensitivity scripts.
4. Reproducibility report template.

## Acceptance Criteria

- Each published claim maps to one or more reproducible experiments.
- Sensitivity analysis available for all headline conclusions.
